# 🧭 Navigation System

A navigation system project that provides:
- Route planning and shortest path calculation
- Map visualization

## 🚀 Features
- Shows the Direction of the Shortest Path
- Find shortest path using algorithms 
- Modular and extensible architecture

## 🛠️ Tech Stack
- C++ 
- Algorithms: Dijkstra
